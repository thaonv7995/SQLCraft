# Development Environment

## 1. Minimum Tooling
- Node.js LTS
- pnpm or npm
- Docker
- PostgreSQL client tools
- Redis
- Makefile or task runner optional

## 2. Recommended Workflow
- run metadata DB and Redis in Docker
- run web/api/worker locally
- run sandbox containers through local Docker
- use seeded demo content

## 3. Useful Dev Scripts
- migrate up
- seed demo data
- start all services
- create sample sandbox
- run integration tests
