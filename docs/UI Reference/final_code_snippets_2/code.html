System Administration

security

Role Inheritance Map

Recursively fetch all groups and nested roles for a specific user. Crucial for debugging access control issues in complex architectures.

WITH RECURSIVE x AS (
  SELECT member, roleid
  FROM pg_auth_members
  WHERE member = 'current_user'
  UNION
  SELECT m.member, m.roleid
  FROM pg_auth_members m
  JOIN x ON m.member = x.roleid
)
SELECT roleid FROM x;

Table Size Overview

Displays the exact disk footprint of each table including its indexes and toasted data. Useful for storage capacity planning.

SELECT relname AS "Table",
  pg_size_pretty(
    pg_total_relation_size(relid)
  ) AS "Size"
FROM pg_catalog.pg_statio_user_tables
ORDER BY 
  pg_total_relation_size(relid) DESC;
